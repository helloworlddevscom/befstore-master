== Description ==

This plugin will add an Advanced Reports section under Downloads, which you can use to create your own reports.

All reports you have created will appear in the Reports section of Easy Digital Downloads, in a new tab named Advanced Reports.

This plugin has no settings page, and everything is managed for each individual report.
